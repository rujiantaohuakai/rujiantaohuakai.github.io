在vscode中配置C++环境(Win11)

TOTURIAL BEGIN

参考CSDN：
VS Code 配置 C/C++ 编程运行环境（保姆级教程）
【Tools】如何在VSCode上使用C++ 保姆教程

1. 安装vscode

进入vscode官网下载安装包安装vscode。

以下全部勾选即可。

2. 安装C/C++编译器（MinGW-w64）

下载MinGW-w64安装包：
这里通过github下载MinGW-w64安装包，下载地址：MinGW-w64
点开后选择最新日期

找到前缀为 msys2-x86_64的.exe文件下载(后面的-20250622是日期)，下图为倒数第五个

双击运行安装程序

可以直接用默认安装路径

一直点下一步，直到安装完毕，点击finish

在弹出的MSYS2终端窗口中输入以下命令，按回车键

出现以下界面后，直接按回车键，默认接受所有的按转包

当系统提示是否继续安装时，输入y并回车

全部安装完毕后，关闭终端窗口

3. 配置C/C++环境变量

打开安装MSYS2的目录，先找到 ucrt64文件夹并进入，再找到 bin文件夹并进入，然后在地址栏中，复制路径

如果一开始用默认路径，那路径就是 C:\msys64\ucrt64\bin

在搜索框输入 编辑系统环境变量并打开，然后点击下面的 环境变量(N)...

找到用户变量的 Path，双击编辑

在弹出的窗口，点击 新建，输入刚才复制的路径，然后点击确定

之后逐层窗口点击确定即可

最后做以下测试，按下 Win+R打开运行框，输入 cmd并回车
依次输入以下命令

如果出现以下信息，说明C/C++de的编译环境安装成功

4. vscode拓展

  1. 汉化
  2. C/C++插件

第一个为必备，安装好后，Ctrl+Shift+P， 选择第一个

C++使用g++.exe, C使用gcc.exe

IntelliSense模式，第一个选择 windows-gcc-x64

  1. Code Runner插件

安装好后，左下角进入设置

在拓展中找到Code Runner，关闭Show Execution Message

5. 编译运行C++程序

使用Run Code运行可以让输出更简洁
